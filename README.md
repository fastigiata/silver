## About Routes

| route                   | desc                     |  |
|-------------------------|--------------------------|--|
| `/lists`                | `list` 列表                |  |
| `/list/create`          | 新建 `list`                |  |
| `/list/{listId}/view`   | 查看 `list` (包含 `item` 列表) |  |
| `/list/{listId}/modify` | 编辑 `list`                |  |
| `/item/create`          | 新建 `item`                |  |
| `/item/{item}/view`     | 查看 `item`                |  |
| `/item/{item}/modify`   | 编辑 `item`                |  |
